# 远程调用终端命令

## 简介

写此程序的目的是为了方便手机或其他设备快速远程调用终端命令

有了它之后，你将可以随时通过命令远程操作你的计算机

- 在被窝里远程关机（写此程序的主要目的...QAQ）
- 在外面打开你想打开的任何程序
- 远程执行写好的脚本
- ......

## 实现原理

利用spring boot处理http请求，`java.lang.Runtime`和`java.lang.Process`实现调用cmd命令

## 使用方法

### 支持的操作系统

- [x] Windows
- [ ] Mac（未完善实现类）
- [ ] Linux（未完善实现类）

### 目前支持的快捷命令

- 测试连通性
  - [测试网络是否通畅](#ping)
- 基础命令
  - [执行cmd命令](#execute)
- 关机类
  - [定时关机](#shutdownInMinute)
  - [马上关机](#shutdownNow)
  - [取消定时关机任务](#shutdownCancel)

#### 测试连通性

##### <a id="ping">测试网络是否通畅</a>

**描述**

测试网络是否通畅

**请求类型**

GET

**请求URL**

/cmd/ping

**返回参数**

| 参数类型 | 描述              |
| :------- | ----------------- |
| string   | success：网络通畅 |

#### 基础命令

##### <a id="execute">执行cmd命令</a>

**描述**

执行指定的cmd命令

**请求类型**

GET

**请求URL**

/cmd

**请求参数**

| 名称 | 类型   | 是否必须 | 描述           |
| :--- | :----- | :------- | :------------- |
| cmd  | string | 是       | 将要执行的命令 |

**返回参数**

| 参数类型 | 描述       |
| :------- | :--------- |
| string   | 输出的内容 |

#### 关机类

##### <a id="shutdownInMinute">定时关机</a>

**描述**

指定时间后关机（分钟为单位）

**请求类型**

GET

**请求URL**

/cmd/shutdown/{minute}

**返回参数**

| 参数类型 | 描述       |
| :------- | :--------- |
| string   | 输出的内容 |

##### <a id="shutdownNow">马上关机</a>

**描述**

马上关闭计算机

**请求类型**

GET

**请求URL**

/cmd/shutdownNow

**返回参数**

| 参数类型 | 描述       |
| :------- | :--------- |
| string   | 输出的内容 |

##### <a id="shutdownCancel">取消定时关机任务</a>

**描述**

取消已经下达的定时关机命令

**请求类型**

GET

**请求URL**

/cmd/shutdownCancel

**返回参数**

| 参数类型 | 描述       |
| :------- | :--------- |
| string   | 输出的内容 |

### 关于网络

在同一局域网下可以利用局域网ip来直接请求；

否则可以利用内网穿透技术或者公网ip做端口映射，此处不再展开。
